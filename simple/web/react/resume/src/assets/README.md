# Resume Assets

This directory contains static assets for the resume application, including images, icons, and other media files.

## Overview

Assets in this directory include:
- Profile photos and avatars
- Company and organization logos
- Icons and illustrations
- Background images
- Font files and custom typography

## Asset Structure

### Images
- `profile/` - Profile and personal photos
- `companies/` - Company logos and branding
- `icons/` - Custom icons and illustrations
- `backgrounds/` - Background patterns and images

### Media
- `fonts/` - Custom font files
- `animations/` - CSS animations and transitions
- `documents/` - Downloadable documents (PDF, etc.)

## Usage Guidelines

- Optimize images for web performance
- Use appropriate file formats (WebP, JPEG, PNG)
- Include descriptive alt text for accessibility
- Follow naming conventions for consistency
- Keep file sizes reasonable for fast loading

## File Naming

Use kebab-case naming:
- `profile-photo.jpg`
- `company-logo-acme.png`
- `icon-skills.svg`
- `bg-pattern-dots.png`
