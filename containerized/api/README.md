# Containerized API Demos

This directory contains API demonstrations that run in containers using development containers.

## Overview

These demos showcase:

- RESTful API development
- Containerized development environments
- API design patterns and best practices
- Integration with databases and services

## Getting Started

Each API demo will include:

- Development container configuration
- API implementation (Express, Fastify, etc.)
- Database integration examples
- API documentation and testing
