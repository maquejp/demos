# Demos Project

A collection of demos showcasing various technologies and patterns.

## Structure

```text
demos/
├── simple/              # Straightforward demos (no DB, no API)
│   ├── web/            # Web frontend demos
│   ├── cli/            # Command line interface demos
│   └── api/            # Simple API demos
├── containerized/       # Demos using devcontainers
│   ├── web/            # Containerized web apps
│   ├── api/            # Containerized APIs
│   └── db/             # Database-integrated demos
├── shared/             # Shared resources
│   ├── templates/      # Project templates
│   ├── components/     # Reusable components
│   └── utils/          # Common utilities
├── docs/               # Documentation
└── scripts/            # Build and utility scripts
```

## Getting Started

Each demo has its own README with specific instructions.

## Development

- Use VS Code workspace for multi-demo development
- Devcontainers are configured for containerized demos
- Git tracks changes across all demos
- See individual demo folders for details
