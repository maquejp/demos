# Demos

## General Info

When adding a project do not forget to update the root README.md file to include it in the list of demos.

When adding a project do not forget to update the root package.json file to include any new scripts or dependencies needed for the new project. And also add the new project to the "workspaces" array. This request the call to `npm install` to install dependencies for all projects in the monorepo.
